Features available from anywhere

 

From within SimPE at any time:

From the application Menu Bar                     

Tools      
  | PJSE    
    | Help Brings up the contents page for PJSE-specific help
    | Refresh File Table Rereads files from disk.   Useful for instance if you are updating files in two instances of SimPE at the same time and wish PJSE to reread the information from them.  PJSE  keeps a filetable cache separately from the SimPE application as a whole, so refreshing one does not refresh the other.   The PJSE filetable affects the interpreting of things like BHAV names and strings for display in the PJSE editors.
    | Object Finder Brings up a dialog where you type in either the GUID or part of the name (OBJD name) of the object you want to find and it lists any matches found within files in the PJSE filetable.  You can copy text from the results to paste into other input boxes such as the SimPE group filter.
       
Extra      
  | Preferences    
    | Custom  
    | PJSE Filetable Preferences. Choose whether to load it at startup of SimPE or only when you first go into a PJSE plugin.  You won't want this loading at startup if most of your work is with non-PJSE plugins such as recolouring.

.

From within any PJSE resource editor plugin:

Buttons on the top border of each editor:

RFT Refresh filetable now
Help Opens Help contents for PJSE
